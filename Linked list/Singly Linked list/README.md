### Singly Linked list -C++

This project implements a Singly Linked List in C++ using pointers

Each node contains:

* An value
* A pointer to the next Node

### DATA STRUCTURE

* Algorithm: **LINKED LIST**

A singly linked list is a linear data structure where each element (node) points to the next element in the list.

### COMPLEXITY ANALYSIS

Insertion:

* addHead: O(1) in best case
* addTail: O(n) in worst case
* addBefore: O(n) in worst case
* addAfter: O(n) in worst case



Deletion:

* deteleHead: O(1) in best case
* deleteTail: O(n) in worst case
* deleteX: O(n) in worst case 



Other operations:

count and check values: O(n) in worst case.
