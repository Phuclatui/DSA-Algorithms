### Graph Coloring -C++

### Features:

* Vertex coloring using Greedy algorithm
* Print coloring result for each vertex

### DATA STRUCTURE

* Algorithm: **Greedy**
* Graph representation: Adjacency list

### COMPLEXITY ANALYSIS

O(V^2)



Where:

V = number of vertices



###### **Description:**

Graph coloring is commonly used in scheduling problems such as:

* Exam scheduling
* Sports tournament scheduling
* Competition scheduling
* Resource allocation problems

The goal is to assign colors to vertices so that no two adjacent vertices share the same color.
