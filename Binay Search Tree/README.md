### MINI MANAGE DICTIONARY-C++

### Features:

* ADD WORD
* SEARCH WORD
* DELETE WORD

### DATA STRUCTURE

* HASH TABLE: **SEPARATE CHAINING**
* METHOD OF HANDLING COLLISION: **SELF-BALANCING BST (AVL)**

### COMPLEXITY ANALYSIS

Time complexity:

* Best case: O(1)
* Average case: O(1 + logk), k is load factor
* Worst case: O(logn)

Space complexity

* O(logn)
