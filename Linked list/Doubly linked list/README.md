### Doubly Linked list -C++

This project implements a Doubly Linked List in C++ using pointers

Each node contains:

* An value
* A pointer to the next Node
* A pointer to the previous Node

### DATA STRUCTURE

* Algorithm: **LINKED LIST**

### COMPLEXITY ANALYSIS

Insertion:

* addHead: O(1) in best case
* addTail: O(1) in worst case
* addBefore: O(n) in worst case
* addAfter: O(n) in worst case



Deletion:

* deteleHead: O(1) in best case
* deleteTail: O(1) in worst case
* deleteX: O(n) in worst case



Other operations:

count and check values: O(n) in worst case.
