### SHOPPING CART-C++

### Features:

* ADD ITEM TO CART
* DELETE ITEM
* DISPLAY CART
* CALCULATE TOTAL PRICE

### DATA STRUCTURE

* HASH TABLE
* METHOD OF HANDLING COLLISION: **COALESCED CHAINING**

### COMPLEXITY ANALYSIS

Time complexity:

* Best case: O(1)
* Average case: O(1+ k), k is load factor
* Worst case: O(n)	

Space complexity

* O(n)
