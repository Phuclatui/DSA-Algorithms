### FINDING SHORTEST PATH-C++

### Features:

* Find the shortest path from a single source vertex
* Get the shortest path distance
* Print the shortest path

### DATA STRUCTURE

* Algorithm: **Dijkstra**
* Graph representation: Adjacency list

### COMPLEXITY ANALYSIS

O(E log V)

E: the number of vertexs

log V: Heapify in queue



**Description**

This program implements Dijkstra's algorithm to compute the shortest paths from a starting vertex to all other vertices in a weighted graph.

The graph is represented using an adjacency list, and a priority queue (min-heap) is used to efficiently select the next vertex with the smallest distance.
