### Minimum Spanning Tree -C++

### Features:

* Connect all vertices without cycles
* Minimize the total cost of connections

### DATA STRUCTURE

* Algorithm: **Kruskal's Algorithm**
* Graph representation: Edge list

### COMPLEXITY ANALYSIS

O(ElogE)



sort: O(ElogE)
Comparision: O(E)

O(ElogE + E) = O(ElogE)
